# Single Page Responsive Custom Screen Design

## Ruby Version
ruby 2.3.0p0

## Rails Version
rails 4.2.0

## Database
MySQL


## How to Setup Application

* mkdir project
* cd project
* git clone git@github.com:PoornimaC/single_page_custom_screen_design.git
* cd single_page_custom_screen_design
* bundle install
* bundle exec rake db:setup
